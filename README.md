nkdash
=========================
Nkdash is a javascript library that includes very useful methods.

# Supported node version
- 7
- 6
- 5

# To use
```
npm install nkdash
```

```
import {isNumber} from 'nkdash'

if(isNumber('1200')) {
  console.log("Success!")
}
```

# Methods
  * pluck
```
  const collection = [
    {a: 1, b: 11, c: 111},
    {a: 2, b: 22, c: 222},
    {a: 3, b: 33, c: 333},
  ]
```
